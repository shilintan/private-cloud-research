# SSH证书登录

## 中控机

生成ssh证书文件

```
ssh-keygen

openssl rsa -in /root/.ssh/id_rsa -outform pem > /root/.ssh/id_rsa.pem
chmod 700 /root/.ssh/id_rsa.pem
# 下载文件 /root/.ssh/id_rsa   /root/.ssh/id_rsa.pub ~/.ssh/id_rsa.pem
```

## 服务器

加载ssh证书文件

```
# 上传id_rsa.pub到服务器
mv id_rsa.pub /root/.ssh/id_rsa.pub
cat /root/.ssh/id_rsa.pub >> /root/.ssh/authorized_keys
chmod 700 /root/.ssh/authorized_keys
```

设置ssh

```
sed -i '/PasswordAuthentication/d'  /etc/ssh/sshd_config
sed -i '/PermitRootLogin/d'         /etc/ssh/sshd_config


cp /etc/ssh/sshd_config sshd_config-bak

cat >> /etc/ssh/sshd_config <<EOF
PermitRootLogin yes
PasswordAuthentication no
RSAAuthentication yes
PubkeyAuthentication yes
PermitRootLogin yes
EOF

systemctl restart sshd
```

# 磁盘

## 磁盘清理

```
wipefs --all --backup /dev/sdb
wipefs --all --backup /dev/sdc
```

## 创建raid1

```
apt install -y mdadm
mdadm -V

lsblk -l
mdadm --create --force --verbose /dev/md0 --level=1 --raid-devices=2 /dev/sdb /dev/sdc
```

## 创建lvm

```
apt install -y lvm2
lvmdiskscan

lsmod | grep dm
modprobe dm-snapshot
lsmod | grep dm

lvmdiskscan
pvcreate -ff /dev/md0
vgcreate lvmvg /dev/md0
```

## 挂载磁盘到目录

```
fdisk -l
mkfs.ext4 /dev/vdb
mkdir /data
mount /dev/vdb /data
df -h /data
echo /dev/vdb /data ext4 defaults 0 0 >> /etc/fstab
reboot
```
